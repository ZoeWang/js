扩展选项卡导航

这个插件扩展 选项卡导航组件 添加tabbable领域.

Usage

通过JavaScript使tabbable标签(需要单独激活每个选项卡):

Copy
$('#myTab a').click(function (e) {
  e.preventDefault()
  $(this).tab('show')
})

你可以激活各个标签，用下面的几种方法:

Copy
$('#myTab a[href="#profile"]').tab('show') // Select tab by name
$('#myTab a:first').tab('show') // Select first tab
$('#myTab a:last').tab('show') // Select last tab
$('#myTab li:eq(2) a').tab('show') // Select third tab (0-indexed)

标记

你可以激活一个标签或药丸导航没有编写任何JavaScript只需指定 data-toggle="tab" or data-toggle="pill" 在一个元素上。 添加 nav and nav-tabs classes 到 tab ul 将应用 Bootstrap tab styling, 同样的添加 nav and nav-pills classes 将应用 pill styling.

<!-- Nav tabs -->
<ul class="nav nav-tabs" role="tablist">
  <li class="active"><a href="#home" role="tab" data-toggle="tab">Home</a></li>
  <li><a href="#profile" role="tab" data-toggle="tab">Profile</a></li>
  <li><a href="#messages" role="tab" data-toggle="tab">Messages</a></li>
  <li><a href="#settings" role="tab" data-toggle="tab">Settings</a></li>
</ul>

<!-- Tab panes -->
<div class="tab-content">
  <div class="tab-pane active" id="home">...</div>
  <div class="tab-pane" id="profile">...</div>
  <div class="tab-pane" id="messages">...</div>
  <div class="tab-pane" id="settings">...</div>
</div>

褪色的效果

标签淡入, 添加 .fade 给每一个 .tab-pane. 第一个选项卡窗格也必须有 .in 在初始内容适当褪色。

Copy
<div class="tab-content">
  <div class="tab-pane fade in active" id="home">...</div>
  <div class="tab-pane fade" id="profile">...</div>
  <div class="tab-pane fade" id="messages">...</div>
  <div class="tab-pane fade" id="settings">...</div>
</div>

方法

$().tab

激活一个标签容器元素和内容. Tab 应该有一个 data-target 或是一个 href 针对一个容器在DOM节点。

Copy
<ul class="nav nav-tabs" role="tablist" id="myTab">
  <li class="active"><a href="#home" role="tab" data-toggle="tab">Home</a></li>
  <li><a href="#profile" role="tab" data-toggle="tab">Profile</a></li>
  <li><a href="#messages" role="tab" data-toggle="tab">Messages</a></li>
  <li><a href="#settings" role="tab" data-toggle="tab">Settings</a></li>
</ul>

<div class="tab-content">
  <div class="tab-pane active" id="home">...</div>
  <div class="tab-pane" id="profile">...</div>
  <div class="tab-pane" id="messages">...</div>
  <div class="tab-pane" id="settings">...</div>

  })

事件

Event Type Description
show.bs.tab 该事件在标签页显示时触发，但是必须在新标签页被显示之前。分别使用 event.target and event.relatedTarget 来定位到激活的标签页和前一个激活的标签页。.
shown.bs.tab 该事件在标签页显示时触发，但是必须在某个标签页已经显示之后。分别使用event.target and event.relatedTarget 来定位到激活的标签页和前一个激活的标签页。.
Copy
$('a[data-toggle="tab"]').on('shown.bs.tab', function (e) {
  e.target //激活的标签页
  e.relatedTarget //前一个激活的标签页
})
