Collapse 折叠 collapse.js

About

得到基本风格和灵活的支持可折叠的组件(比如手风琴和导航)。

Plugin 依赖

Collapse 需要 the transitions plugin 包含在你的Bootstrap版本里 .

您还可以使用插件没有accordion的标记。让一个按钮切换另一个元素的扩展和收缩。

<button type="button" class="btn btn-danger" data-toggle="collapse" data-target="#demo" aria-expanded="true" aria-controls="demo">
  simple collapsible
</button>

<div id="demo" class="collapse in">...</div>

使展开/折叠控制访问

确定添加 aria-expanded 去控制这个元素. 这个属性显式地定义了可折叠的元素的当前状态屏幕阅读器和类似的辅助技术. 如果折叠元素在默认情况下是关闭的, 它应该有一个值 aria-expanded="false". 如果你想设置默认折叠面板时打开的，给打开的元素添加 in class, 设置 aria-expanded="true" 控制. 插件会自动切换这个属性基于折叠元素是否已被打开或关闭.

另外, 如果你的控制元素是一个可折叠的元素– i.e. the data-target 属性是指向一个 id 选择器 – 你可以添加一个额外的 aria-controls 属性控制元素, 包含 这个 id 所在的折叠元素. 现代屏幕阅读器和类似的辅助技术利用这个属性直接为用户提供附加的快捷键来导航可折叠元素本身.

Usage

The collapse plugin 利用 几个 classes处理繁重的生存过程:

  • .collapse 隐藏内容
  • .collapse.in 显示内容
  • .collapsing 当转换开始时添加它, 当转换结束后移除它。

这些class都在 component-animations.less里面.

通过 data 属性

只需要添加 data-toggle="collapse" 和一个 data-target 给一个元素去自动标记控制一个折叠的元素. 这个data-target 属性 接受一个 CSS 选择器给折叠应用. 确定添加一个 class collapse 给要折叠的元素.如果你想让他默认是打开的, 添加一个额外的 class in.

添加一个就像组管理的折叠元素, 添加这个 data 属性 data-parent="#selector". 参考的例子在上面.

通过 JavaScript 调用

手动启用:

$('.collapse').collapse()

选项

选项能通过 data 属性或 JavaScript. 如果是 data 属性, 附加选项名称给 data-, 就像 data-parent="".

Name type default description
parent selector false 如果提供了一个选择器,当前可折叠项目显示时，在其指定父元素下的所有其他可折叠的元素将被关闭. (类似的传统的手风琴行为 - 他依赖于 panel class)
toggle boolean true 切换可折叠元素时调用

方法

.collapse(options)

激活你的内容作为一个可折叠的元素。接受一个可选的选项 object. 

$('#myCollapsible').collapse({
  toggle: false
})

.collapse('toggle')

切换可折叠元素来显示或隐藏。.

.collapse('show')

显示一个可折叠元素.

.collapse('hide')

隐藏一个可折叠元素.

事件

下表列出了Bootstrap's折叠（Collapse）插件中要用到的事件。这些事件可在函数中当钩子使用。.

Event Type Description
show.bs.collapse 当调用 show 事件时，立即触发该事件.
shown.bs.collapse 当折叠元素对用户可见完成时触发该事件（将等待 CSS 过渡效果完成）。.
hide.bs.collapse 当调用 hide 事件时，立即触发该事件.
hidden.bs.collapse 当折叠元素对用户隐藏完成时触发该事件（将等待 CSS 过渡效果完成）。. 
$('#myCollapsible').on('hidden.bs.collapse', function () {
  // do something…
})
