工具提示 tooltip.js

当您想要描述一个链接的时候，工具提示（Tooltip）就显得非常有用。工具提示（Tooltip）插件是受 Jason Frame 写的 jQuery.tipsy 的启发。工具提示（Tooltip）插件做了很多改进，例如不需要依赖图像，而是改用 CSS 实现动画效果，用 data 属性存储标题信息。

工具提示在零长度的标题时显示不出来。

可选 功能

由于性能关系， the Tooltip 工具提示 and Popover 弹出框 data-apis 是可选的, 意味着 你必须自己初始化他们.

工具提示在按钮组button groups和输入组input groups需要特殊设置

当我们使用tooltips 在一个 .btn-group or an .input-group元素上,你必须指定选项 container: 'body' (以下有记录),以避免不必要的副作用(如元素日益广泛和/或工具提示触发时失去其圆角)。

不要试图隐藏元素显示工具提示

调用 $(...).tooltip('show') 当目标元素 display: none; 会导致错误定位的工具提示。

工具提示在禁用元素需要包装器元素

添加tooltip在 disabled or .disabled 元素, 把这个元素放到一个<div> 中，应用这个 tooltip 在 <div> 上而不是有disabled的元素.

Usage

工具提示（Tooltip）插件根据需求生成内容和标记，默认情况下是把工具提示（tooltip）放在它们的触发元素后面。

触发tooltip 通过 JavaScript:

$('#example').tooltip(options)

标记

所需的工具提示仅仅是一个标记data 属性 和 title 在 HTML 你希望有 tooltip的元素上. 生成的标记工具提示是相当简单的,但需要一个位置(默认时, 插件设置的是 top 位置).

多重线链接

有时你想添加一个提示一个超链接包装多个行. 工具提示插件的默认行为是横向和纵向中心。添加 white-space: nowrap; 到你的锚点,以避免这个问题。

Copy
<!-- HTML to write -->
<a href="#" data-toggle="tooltip" title="Some tooltip text!">Hover over me</a>

<!-- Generated markup by the plugin -->
<div class="tooltip top" role="tooltip">
  <div class="tooltip-arrow"></div>
  <div class="tooltip-inner">
    Some tooltip text!
  </div>
</div>

选项

选项能够通过 data 属性 or JavaScript. 对于 data 属性, 加入选项名称 data-, as in data-animation="".

Name Type Default Description
animation boolean true 应用一个CSS褪色过渡到工具提示
container string | false false

将工具提示附加到一个特定的元素. Example: container: 'body'. 这个选项特别有用,因为它允许您触发元素位置附近的文档流中的工具提示——这将防止工具提示浮离触发元素在一个窗口的大小。
delay number | object 0

延迟显示和隐藏工具提示(ms)并不适用于手动触发类型

如果提供一个数字,延迟应用hide/show

对象结构 是: delay: { "show": 500, "hide": 100 }
html boolean false 插入HTML工具提示. If false, jQuery's text 方法将用于将内容插入到DOM。如果你担心XSS攻击请使用文本。
placement string | function 'top'

如何定位工具提示 - top | bottom | left | right | auto.
当 "auto" 指定,它会动态地调整工具提示。For example, 如果 placement 是 "auto left", 工具提示将尽可能的显示在左边, 在情况不允许的情况下它才会显示在右边。.

当一个函数是用来确定位置,它的第一个参数是调用DOM节点 ，第二个参数是触发元素DOM节点 . 这个 this 是工具提示的实例。
selector string false 如果提供了一个选择器, 工具提示对象将委派给指定的目标. 在实践中,这是用于启用动态HTML内容提示补充道。 See this and an informative example.
template string '<div class="tooltip" role="tooltip"><div class="tooltip-arrow"></div><div class="tooltip-inner"></div></div>'

基于HTML创建工具提示时使用。

The tooltip's title 将注入 .tooltip-inner.

.tooltip-arrow 将成为the tooltip's arrow.

最外层的包装器元素应该有 .tooltip class.
title string | function ''

如果未指定 title 属性，则 title 选项是默认的 title 值.

如果给出一个函数, 它将被称为 this 参考设置工具提示附加的元素。
trigger string 'hover focus' 定义如何触发工具提示： click| hover | focus | manual。您可以传递多个触发器，每个触发器之间用空格分隔。
viewport string | object { selector: 'body', padding: 0 }

保持工具提示这个元素的范围内. Example: viewport: '#viewport' or { "selector": "#viewport", "padding": 0 }

事件

下表列出了工具提示（Tooltip）插件中要用到的事件。这些事件可在函数中当钩子使用。

Event Type Description
show.bs.tooltip 当调用 show 实例方法时立即触发该事件。
shown.bs.tooltip 当工具提示对用户可见时触发该事件（将等待 CSS 过渡效果完成）。
hide.bs.tooltip 当调用 hide 实例方法时立即触发该事件。
hidden.bs.tooltip 当工具提示对用户隐藏时触发该事件（将等待 CSS 过渡效果完成）。
Copy
$('#myTooltip').on('hidden.bs.tooltip', function () {
  // do something…
})
